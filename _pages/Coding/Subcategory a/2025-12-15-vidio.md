* [문제](https://school.programmers.co.kr/learn/courses/30/lessons/12909)

풀이 과정
	주어진 오프닝 시작,끝, 현재 시간, 종료 시간을 int 형식으로 변환한뒤
	 명령을 실행후 string 형식으로 변경
코드
```cpp
#include<string>
#include <iostream>

using namespace std;

bool solution(string s)
{
    int a = s.length();
    int top = 0;
    for (int i = 0; i < a; i++) {
        if (s[i] == '(') {
            top++;
        }
        else {
            top--;
            if (top == -1) {
                return false;
            }
        }
            
    }
    if(top != 0){
        return false;
    }
    return true;
}
```
