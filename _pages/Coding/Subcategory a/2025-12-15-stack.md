* [문제](https://school.programmers.co.kr/learn/courses/30/lessons/12909)

풀이 방법
	굳이 스택을 구현 할 필요 없이 top만 구현해서
	 스택이 비었는데 pop할경우 false
	 스택이 남았는데 끝났을 경우 false
	 나머지는 true로 한다
코드
```cpp
#include<string>
#include <iostream>

using namespace std;

bool solution(string s)
{
    int a = s.length();
    int top = 0;
    for (int i = 0; i < a; i++) {
        if (s[i] == '(') {
            top++;
        }
        else {
            top--;
            if (top == -1) {
                return false;
            }
        }
            
    }
    if(top != 0){
        return false;
    }
    return true;
}
```
